# Lily's Little Fantasies
[![unity](https://img.shields.io/badge/unity-2019.2.15-blue)](https://unity3d.com/fr/get-unity/download)
[![visual-studio](https://img.shields.io/badge/visualstudio-2019-purple)](https://visualstudio.microsoft.com/fr/vs/)
[![aseprite](https://img.shields.io/badge/aseprite-1.2.16-lightgrey)](https://www.aseprite.org/)

Game Jam Project made in 48 hours during the Geta Game Jam 2019.

You can play the game [here](https://raychop.itch.io/lilys-little-fantasies).

## Description
Travel upon a great journey to fight Lily’s demons, using the imagination of this cute little girl to find your way into her nightmares and repaint them into dreams !

## Tools
Unity 2018.3.7 / Aseprite / Bfxr.
