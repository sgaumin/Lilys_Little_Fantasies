# Lily's Little Fantasies
Game Jam Project made in 48 hours during the Geta Game Jam 2019 (Unity 3D, Aseprite, C#).

You can play the game [here](https://raychop.itch.io/lilys-little-fantasies).

## Description
Travel upon a great journey to fight Lily’s demons, using the imagination of this cute little girl to find your way into her nightmares and repaint them into dreams !

## Tools
Unity 2018.3.7 / Aseprite / Bfxr.
